Overview

This Odoo module allows users to change report font size to make it more readable or save space in the page.

Features

  • Easy integration with Odoo
  • Free font size edit
  • Customizable from the interface

Usage

  1. After installation, navigate to Settings > General Settings, in companies section click on "Configure Document Layout".
  2. Edit the "Font size" field to the desired size.
